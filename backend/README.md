# Tripleten web_project_around_express

Este proyecto es un servidor backend desarrollado con Node.js y Express que proporciona una API sencilla para gestionar usuarios y tarjetas.
-Crear, obtener y actualizar usuarios.
-Crear, obtener, eliminar y dar like a tarjetas.

# Tecnologias

-Node.js
-Express.js
-JavaScript (ES Modules)
-Nodemon
